# From Zero to Deployed: Build a Full-Stack AI Blogging Platform

A complete beginner-to-builder guide covering TypeScript, React, Tailwind CSS, Node.js, Express, and MongoDB — by rebuilding this project from scratch.

## How to use this book

Each chapter builds on the last. Read in order. By the end, you'll have recreated every file in this project by hand, using nothing but VS Code and your terminal.

Code blocks with a filepath comment (`// src/foo.ts`) show you exactly where the code lives in the final project. **Type every example yourself** — copying builds muscle memory.
